# myjs-common 帮助文档

### 运行环境
 - nodejs version: v12.10.0
 - npm version: 6.10.3


### 项目使用
```bash
    # 修订版本变更
    npm run patch
    # 子版本变更
    npm run minor
    # 主版本变更
    npm run major
    # 说明: 版本号变更可直接修改 package.json 文件的 version 属性


    # 发布 包
    npm run p

    # 执行测试脚本
    npm run tdu
    npm run tf
    npm run thm
```
